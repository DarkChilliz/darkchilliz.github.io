Cookie usage?
This website uses cookies to store the options you modify in the settings menu. That's it. Also, if you are blocking cookies you might have issues loading chat.
Keyboard / mouse shortcuts?
- F11 to fullscreen with chat
- Hold Shift to add streams without chat
- Hold Ctrl to close menu after an action
- Hold Shift when reloading a stream for it to remember its volume, quality, etc.
- Scrollwheel to change dropdown options
Which services are supported?
Mainly Twitch but also Mixer and Youtube. Twitch channel names can be added as is, but for Mixer and Youtube you add the stream or video link instead (or the stream/video name preceded by "mixer=" or "v=" respectively. For example: mixer=monstercat v=0d0WB4sj7OE ).
Which links are supported?
- Any Twitch stream, clip, or video link
(including video "collections").
- Any Mixer stream or video link
(clips hopefully soon).
- Any Youtube stream, video or playlist link.
- Links from other multi websites
Please message me with links that are not working.
Can I add multiple channels at once?
Yes! Just separate them by spaces, comas or forwardslashes. Add links one at a time.
Click to play/unmute?
These are new measures taken by modern web browsers to prevent websites from playing loud unwanted audio and wasting your valuable mobile data. Either click the icons to play/unmute the players or change the website permissions on your browser settings.
How do I copy the link to my setup?
Right click the link and select "Copy link address" or "Copy link shortcut". On mobile, tap and hold the link to get the option.
How does drag-and-drop work?
You may drag any one link (from other windows/tabs, like your followed channels page) and drop it anywhere on the website. Any subsequent streams should be added using the menu (which also supports drag-and-drop).
Drag-and-drop not working?
Some (old?) web browsers disable this if you already have streams or chats on the screen.
Time stamps for VODs and videos
TwitchTheater now supports parameters on videos. The time parameter allows you to specify the time stamp where a video should start playing at when added. This allows you to create multi VOD Theater links where all the videos start playing at a different time stamp but in sync with each other. This is supported for Twitch, Mixer, and Youtube videos. The syntax is like so: "&t=11h22m33s" (a different digit for the hours, the minutes, and the seconds. Example: https://youtu.be/vnnf42BQTfo?list=PL2B009153AC977F90&t=6m11s ).
Additional Youtube parameters
TwitchTheater supports additional parameters for Youtube videos. These are: "&loop" (to loop videos or playlists), and "&shuffle" (to shuffle videos on playlists).
Youtube chat issues
Known issues: (1) Chat replay is not yet available for 3rd party websites. (2) There is no difference between a Youtube video link and a Youtube live stream link, so videos might get added with a "video unavailable" chat window.
VODs and clips dont have chat?
Chat replay is a feature that has not yet been made available for embedding.
Stream hosting not working?
For now, you have to add the stream that is being hosted.
Play/pause, mute/unmute not working?
Twitch clips (and sadly Mixer streams and VODs) dont yet allow/support some menu features. These include: play/pause, mute/unmute, volume, and video quality menu controls. If these functions stop working on other players, reload them.
Chat dark mode not working?
This setting will be applied to new chats. Existing chats need to be reloaded. It has no effect on Mixer chats. If it stops working re-enable dark mode on the main website.
Only one chat loading when added?
This setting defaults to disabled to help people with slow connections.
Changing audio not switching chat?
This only works with the menu controls.
How does smart auto-unmute work?
When the only player with audio is removed or goes offline, it will try to auto-unmute a different one. It will also auto-unmute streams that come back online if no other player is playing audio.
Default video quality not working?
This feature currently only works with Twitch streams and VODs (but not clips).
Wrong video quality got set?
The selected video quality was not available on one or more streams. The closest video quality on that player was used instead.
Why wont fullscreen zoom out?
This happens on mobile Chrome. Back out of fullscreen and let the Theater auto-adjust itself to the screen size (after you close the menu).
How to hide the address bar?
Some mobile web browsers allow you to hide the address bar to maximize the screen space available to the webpage (without going fullscreen). To do so, zoom in using your fingers, scroll down, then zoom out.
What is the website's link format?
Add the stream names after the website address, separated by forwardslashes. To specify the chats to load, just add a "/c/" at the end of the URL and then add the chat names after that (also separated by forwardslashes). If the chat parameter is omitted, then the site will add chats based on the stream names. Use an empty chat parameter to prevent chats from getting added. Use it without streams and you will force load a clean session.
How does session saving work?
It will store the streams and chats you are using and will restore them next time you open or reload the page. When loading the page with streams/chats in the URL, it will use that instead (and will store them).
Custom menu color?
Search Google for an "online color picker", make the color of your choice, then copy the color code with a format like: #336699 (with or without the hash character). Dark colors recommended.
